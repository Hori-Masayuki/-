# -
生徒の成績を管理するシステムです。


初めて使用する方はサインアップする必要があります。
サインアップ後、サインインすることが出来ます。

機能は以下の通りです。
1. 新規生徒登録
2. 生徒情報の更新、削除
3. 生徒ごとに成績を登録
4. 成績を一覧表示、成績の削除
